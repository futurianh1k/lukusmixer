## 소개

LUKUS Music Mixer는 AI 기반 음악 스템(STEM) 분리 도구입니다.

### 스템(STEM)이란?
음악을 구성하는 개별 트랙을 의미합니다:
- **Vocals**: 보컬, 노래 소리
- **Drums**: 드럼, 타악기
- **Bass**: 베이스 기타
- **Guitar**: 기타
- **Piano**: 피아노, 키보드
- **Other**: 기타 악기

### 주요 기능
- 음악 파일에서 보컬과 악기 분리
- 최대 14개 스템 분리 (현악기, 금관악기 등)
- 사용자 정의 악기 분리 (커스텀 쿼리)
- 자연어 기반 믹싱 ("보컬 키워줘")
- 스펙트로그램 시각화

## 기본 사용법

### Step 1: 파일 업로드

1. **설정 패널**에서 "Uploaded File" 영역 확인
2. 두 가지 업로드 방법:
   - **드래그 앤 드롭**: 파일을 업로드 영역에 끌어다 놓기
   - **클릭**: 업로드 영역 클릭 후 파일 선택

3. 업로드 완료 후 파일 정보 표시:
   - 파일명
   - 재생 시간
   - 파일 크기

### Step 2: 모델 선택

**Model** 섹션에서 분리 모델 선택:

| 모델 | 스템 수 | 품질 | 속도 | 권장 용도 |
|------|--------|------|------|----------|
| 4 스템 (기본) | 4개 | 좋음 | 빠름 | 빠른 분리 |
| 6 스템 (고급) | 6개 | 높음 | 보통 | 기타/피아노 필요 시 |
| 6 스템 고급 보컬 | 6개 | 최고 | 느림 | 보컬 품질 중요 시 |
| 10 스템 드럼 분리 | 10개 | 높음 | 느림 | 드럼 세부 분리 |
| 14 스템 풀 오케스트라 | 14개 | 높음 | 느림 | 오케스트라 음악 |

### Step 3: 스템 선택

**Select Stems** 섹션에서 분리할 스템 선택:

- 기본적으로 모든 스템 선택됨
- 특정 스템만 필요하면 클릭하여 선택/해제
- **Select All**: 모든 스템 선택

### Step 4: 분리 시작

1. **Split Stems** 버튼 클릭
2. 진행률 표시:
   - 진행 바로 현재 상태 확인
   - 로그 패널에서 상세 진행 상황 확인
3. 처리 시간: 음악 길이에 따라 1~5분 소요

### Step 5: 결과 확인

분리 완료 후 **결과 패널**에서:

1. 각 스템별 카드 표시
2. **재생 버튼**: 스템 미리듣기
3. **스펙트로그램**: 주파수 시각화
4. **볼륨 조절**: 스템별 볼륨 조절

## 스템 분리 모델

가장 빠른 기본 분리 모델

**분리되는 스템:**
- Vocals (보컬)
- Drums (드럼)

**권장 상황:**
- 빠른 분리가 필요할 때
- 보컬만 제거하고 싶을 때
- MR (반주) 만들기

기타와 피아노를 추가로 분리

**분리되는 스템:**
- Vocals, Drums, Bass
- **Guitar** (기타)
- **Piano** (피아노)
- Other

**권장 상황:**
- 기타 솔로 추출
- 피아노 반주 분리
- 밴드 음악 편집

### 6 스템 고급 보컬 — bs_roformer_6s

BS-RoFormer + Demucs 체이닝으로 최고 품질 보컬 분리

**특징:**
- 보컬 분리 품질 최고
- 처리 시간 2배 소요
- 아티팩트 최소화

**권장 상황:**
- 보컬 품질이 가장 중요할 때
- 노래방 MR 제작
- 리믹스/커버 제작

드럼을 세부적으로 분리

**분리되는 스템:**
- **Lead Vocals** (메인 보컬)

- **Kick** (킥 드럼)
- **Snare** (스네어)
- **Toms** (탐)
- **Cymbals** (심벌즈)
- Bass, Guitar, Piano, Other

**권장 상황:**
- 드럼 프로그래밍 참고
- 드럼 교체/편집
- 음악 교육용

---

### 14 스템 풀 오케스트라 — bs_roformer_banquet

Banquet 모델로 오케스트라 악기까지 분리

**분리되는 스템:**
- 10스템 기본 + 추가 4개:
- **Strings** (현악기: 바이올린, 첼로 등)
- **Brass** (금관악기: 트럼펫, 트롬본 등)

- **Synthesizer** (신디사이저)

**권장 상황:**
- 오케스트라/영화 음악
- 클래식 편곡
- 신디사이저 추출

## 커스텀 쿼리 분리

원하는 악기 소리를 직접 업로드하여 분리하는 고급 기능

### 커스텀 쿼리란?

- "이 소리와 비슷한 소리를 찾아서 분리해줘"
- 예: 바이올린 소리 샘플 업로드 → 음악에서 바이올린 분리

### 사용 방법

#### 1. 커스텀 쿼리 모드 활성화

1. **Custom Query Mode** 토글 ON
2. 커스텀 쿼리 관리 패널 표시됨

#### 2. 쿼리 오디오 업로드

1. **쿼리 이름** 입력 (예: "바이올린", "트럼펫")
2. **색상** 선택 (결과 구분용)
3. 쿼리 오디오 파일 업로드
   - 권장 길이: 5~15초
   - 최대 길이: 30초
   - 단일 악기 소리가 명확할수록 좋음

#### 3. 쿼리 선택

업로드된 쿼리 목록에서:
- **체크박스**: 분리에 사용할 쿼리 선택
- **재생 버튼**: 쿼리 미리듣기
- **휴지통**: 쿼리 삭제

#### 4. 통합 분리 실행

**Split Stems + N개 커스텀** 버튼 클릭

결과:
- 기존 모델 스템 (vocals, drums 등)
- \+ 커스텀 쿼리 스템 (바이올린, 트럼펫 등)

### 좋은 쿼리 오디오 예시

| 좋은 예 | 나쁜 예 |
|--------|--------|
| 단일 악기 솔로 연주 | 여러 악기가 섞인 음악 |
| 깨끗한 녹음 | 잡음이 많은 녹음 |
| 5~15초 길이 | 1초 미만 또는 30초 초과 |
| 다양한 음역대 포함 | 단일 음만 반복 |

### 쿼리 오디오 구하기

무료 샘플 사이트:
- [Freesound.org](https://freesound.org) — CC0 라이선스 샘플
- [SampleFocus](https://samplefocus.com) — 악기 샘플
- 직접 녹음한 악기 소리

## 프롬프트 믹싱

자연어로 스템 볼륨을 조절하는 기능

### 사용 방법

1. 스템 분리 완료 후 **믹싱 패널** 확인
2. 프롬프트 입력창에 원하는 조절 입력
3. **Mix** 버튼 클릭
4. 결과 미리듣기 및 다운로드

### 프롬프트 문법

#### 기본 형식
```
[악기] [동작]
```

#### 악기 키워드

| 한국어 | 영어 | 대상 스템 |
|--------|------|----------|
| 보컬, 목소리, 노래 | vocals | Vocals |
| 드럼, 드럼스 | drums | Drums |
| 베이스 | bass | Bass |
| 기타 | guitar | Guitar |
| 피아노, 키보드 | piano | Piano |
| 현악기, 스트링 | strings | Strings |
| 금관악기, 브라스 | brass | Brass |

#### 볼륨 조절 키워드

| 키워드 | 효과 |
|--------|------|
| 키워, 올려, 크게, 강조 | +6dB |
| 조금 키워, 살짝 키워 | +3dB |
| 줄여, 작게, 낮춰 | -6dB |
| 조금 줄여 | -3dB |
| 최대로 크게 | +12dB |
| 음소거, 뮤트, 제거 | 무음 |

#### 구간 지정

| 키워드 | 구간 |
|--------|------|
| 전주, 인트로 | 0~15초 |
| 후주, 아웃트로 | 마지막 30초 |
| 전체, 모두 | 전곡 |
| 1분~2분 | 60초~120초 |

### 프롬프트 예시

#### 단순 조절
```
보컬 키워줘
드럼 줄여
베이스 음소거
```

#### 복합 조절
```
보컬 키워줘
드럼 조금 줄여
기타 크게
```

#### 구간 지정
```
전주에서 드럼 음소거
1분~2분 보컬 강조
아웃트로 피아노 키워
```

### 믹싱 결과

- 실시간 미리듣기
- MP3 다운로드 (320kbps)
- 프롬프트 히스토리 저장

## 결과 다운로드

### 개별 스템 다운로드

각 스템 카드에서:
- **다운로드 버튼** 클릭
- 파일명: `원본파일명_스템명.mp3`

### 전체 다운로드

**Download All** 버튼:
- 모든 스템을 ZIP 파일로 다운로드
- 파일명: `원본파일명_stems.zip`

### 믹싱 결과 다운로드

믹싱 패널에서:
- **다운로드** 버튼 클릭
- 파일명: `원본파일명_mixed.mp3`

### 라이브러리 저장

**Add to Library** 버튼:
- 분리 결과를 라이브러리에 저장
- 나중에 다시 접근 가능

## 자주 묻는 질문

### 일반

**Q: 처리 시간이 얼마나 걸리나요?**

A: 음악 길이와 선택한 모델에 따라 다릅니다:
- 4스템 기본: 1분당 약 30초
- 6스템 고급: 1분당 약 1분
- 14스템: 1분당 약 2분

**Q: 어떤 음악이 가장 잘 분리되나요?**

A: 
- 스튜디오 녹음 품질의 음악
- 악기 구성이 명확한 음악
- 높은 비트레이트 (320kbps+)

**Q: 라이브 녹음도 분리되나요?**

A: 가능하지만 품질이 떨어질 수 있습니다. 관객 소리, 잔향 등이 "Other"에 포함됩니다.

---

### 품질 관련

**Q: 분리된 스템에서 다른 악기 소리가 들려요**

A: AI 분리의 한계입니다. 개선 방법:
- 더 높은 품질 모델 선택 (6스템 고급 보컬)
- 원본 파일 품질 확인
- 믹싱으로 불필요한 소리 줄이기

**Q: 보컬이 깨끗하게 분리되지 않아요**

A: "6스템 고급 보컬" 모델을 사용하세요. BS-RoFormer 기반으로 보컬 분리 품질이 가장 좋습니다.

---

### 커스텀 쿼리

**Q: 커스텀 쿼리가 잘 안 돼요**

A: 쿼리 오디오 품질을 확인하세요:
- 단일 악기만 포함되어 있는지
- 충분한 길이 (5~15초)
- 잡음이 없는지

**Q: 어떤 악기를 커스텀 분리할 수 있나요?**

A: 이론적으로 모든 소리 가능:
- 전통 악기 (가야금, 해금 등)
- 특수 효과음
- 특정 가수의 목소리

---

### 기술적 문제

**Q: 업로드가 안 돼요**

A: 확인 사항:
- 파일 형식: MP3, WAV, FLAC, OGG, M4A
- 파일 크기: 200MB 이하
- 인터넷 연결 상태

**Q: 처리 중 오류가 발생했어요**

A: 
- 페이지 새로고침 후 다시 시도
- 다른 파일로 테스트
- 문제 지속 시 관리자에게 문의

**Q: 결과가 다운로드되지 않아요**

A:
- 팝업 차단 해제 확인
- 다른 브라우저로 시도
- 개별 스템 다운로드 시도

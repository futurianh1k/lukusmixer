# Banquet 레퍼런스 쿼리 오디오

이 폴더의 WAV 파일은 Banquet 쿼리 기반 분리에 사용됩니다.

## 파일 목록
- `strings_query.wav` — 현악기 (바이올린) 레퍼런스
- `brass_query.wav` — 금관악기 (트럼펫) 레퍼런스
- `woodwinds_query.wav` — 목관악기 (플루트) 레퍼런스
- `synthesizer_query.wav` — 신디사이저 레퍼런스

## 품질 향상 팁
합성 오디오 대신 실제 악기 녹음을 사용하면 분리 품질이 향상됩니다.
- Freesound.org (CC0 라이선스): https://freesound.org
- MoisesDB 레퍼런스: https://github.com/moises-ai/moises-db
